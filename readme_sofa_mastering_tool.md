# SOFA Mastering & Extras Tool

A standalone GUI utility for post-processing and mastering SOFA (Spatially Oriented Format for Acoustics) files generated by Mesh2HRTF or other HRTF simulation tools. This tool provides a user-friendly interface for generating production-ready SOFA files and additional acoustic data.

## Features

*   **Drag & Drop Interface:** Easily load `.sofa` files by dragging them into the application.
*   **SOFA Mastering:** Automatically generates four versions of your SOFA file:
    *   44.1 kHz - Diffuse-Field Equalized
    *   44.1 kHz - Non-Diffuse-Field Equalized
    *   48 kHz - Diffuse-Field Equalized
    *   48 kHz - Non-Diffuse-Field Equalized
*   **Diffuse Field HRTF (Extras):** Generates specialized outputs for headphone equalization:
    *   **CSV Exports:** Diffuse-field responses for left, right, and average with optional tilt.
    *   **Frequency Plots:** Visual representations of the HRTF data.
    *   **Customizable Tilt:** Adjust the Diffuse Field tilt (e.g., -0.8 dB per octave) to match specific target curves.
*   **Flexible Output:** Save results in the same folder as the input or a custom directory.

## Prerequisites

Ensure you have the following installed:

1.  **Python 3.10+**
2.  **Required Libraries:**
    ```bash
    pip install customtkinter tkinterdnd2-universal numpy scipy
    ```
    *(Note: `generate_sofa_outputs.py` and `generate_extras.py` must be present in the same directory as the tool.)*

## Usage

1.  Run the mastering tool:
    ```bash
    python sofa_mastering_tool.py
    ```

2.  **Process a SOFA file:**
    *   **Input:** Drag and drop your raw `.sofa` file onto the "Drag & Drop" zone or click to browse.
    *   **Output Destination:** Choose whether to save the new files in the same folder as the input or select a custom path.
    *   **Generate Mastered SOFA:** Click this button to run the mastering script (`generate_sofa_outputs.py`). This creates standardized files compatible with binaural renderers like SPARTA or APL Virtuoso.
    *   **Generate DFHRTF Files:** Adjust the **DF Tilt Amount** slider (default -0.80 dB/oct) and click "Generate DFHRTF Files" to run the extras script (`generate_extras.py`).

## How to use the outputs

*   **Mastered SOFA Files:** Use these in DAW plugins (SPARTA, Virtuoso, etc.) for binaural monitoring. Virtuoso expect the equalized version but SPARTA plugins can use either version.
*   **CSV Extras:** These files can be used to equalize your headphones to your personal HRTF. Take blocked-ear canal measurements of your headphones (on your ears!) and equalize them for generated tilted diffuse-field response. (For use with virtuoso, which already applies a downward tilt above 1 kHz, you may want equalize to untilted DFHRTF.)
